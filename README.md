# webservice
一、启动服务器前准备<br>
　１、导入库。本程序应用到的Python导入库有：<br>
　　　　pyftpdlib<br>
　　I、其中库：pyftpdlib可以使用如下命令安装：<br>
  　　　pip install <库名>　（如：pip install pyftpdlib）<br>
　　　如安装过程中出现下载错误可以进入packages目录，实施本地安装。<br>
二、文件说明。<br>
　ftpserver.py FTP服务器实现 <br>　
　webserver.py WEB服务器实现 <br>
三、启动服务。<br>
　python -m <文件名> <br>
四、关于目录结构说明：<br>
　　\webservice<br>
  　　｜\packages　　所需库目录<br>
